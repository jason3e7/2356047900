## EasyCTF_liar code review
```
m = n ^ 0x58EB29
g[i] = (m * i) ^ f[i]
```
* `g[i] = ((n ^ 0x58EB29) * i) ^ f[i]`  
* `(g[i] ^ f[i]) / i = n ^ 0x58EB29`
  * a xor b = c, c xor a = b, b xor c = a.

Note:
```
(97  ^ 102) / 1 = 7
(115 ^ 125) / 2 = 7
(121 ^ 108) / 3 = 7
(99  ^ 127) / 4 = 7
(116 ^ 87)  / 5 = 7

10110001110101100101001
```
